Python dasturlash tilining asoslarini o'rganamiz

Python nima? Asosiy

Python — bu oson o'rganiladigan, kuchli va universal dasturlash tili. U web-dasturlash, sun'iy intellekt, data analysis (ma'lumotlar tahlili), avtomatlashtirish va boshqa ko'plab sohalarda ishlatiladi. Python sintaksisi sodda va tushunarli.

print() funksiyasi Chiqarish

print() funksiyasi ekranga ma'lumot chiqarish uchun ishlatiladi. Qavs ichiga yozilgan matn yoki son konsolda ko'rsatiladi. Matnni qo'shtirnoq ichiga yozish kerak.

print_example.py 
# Konsolga matn chiqarish
print("Salom, dunyo!")
print("Men Python o'rganayapman")
print(2024)

💡 print() funksiyasi har qanday ma'lumotni ekranga chiqaradi. Matn uchun qo'shtirnoq " " ishlatiladi, sonlar uchun esa shart emas.

input() funksiyasi Kiritish

input() funksiyasi foydalanuvchidan ma'lumot olish uchun ishlatiladi. Dastur to'xtab, foydalanuvchi javob kiritishini kutadi. Kiritilgan ma'lumot matn (string) ko'rinishida saqlanadi.

input_example.py 
# Foydalanuvchidan ma'lumot olish
ism = input("Ismingizni kiriting: ")
print("Salom,", ism)

💡 input() har doim matn (string) qaytaradi. Agar son kerak bo'lsa, int() yoki float() bilan o'zgartirish kerak.

Variable (O'zgaruvchi) Saqlash

O'zgaruvchi — bu ma'lumotni saqlash uchun "quti" kabi narsa. Unga nom beriladi va ichiga qiymat joylashtiriladi. Keyinchalik bu qiymatdan foydalanish mumkin.

variable_example.py 
# O'zgaruvchilarga qiymat berish
ism = "Ali"
yosh = 25
maosh = 1500000

# O'zgaruvchilarni chiqarish
print("Ism:", ism)
print("Yosh:", yosh)
print("Maosh:", maosh)

💡 O'zgaruvchi nomi lotin harflari bilan boshlanishi kerak. Bo'sh joy o'rniga _ ishlatiladi. Masalan: user_name

Oddiy hisob-kitoblar Arifmetika

Pythonda matematika amallari juda oson bajariladi: + qo'shish, - ayirish, * ko'paytirish, / bo'lish, % qoldiq.

calculation_example.py 
# Arifmetik amallar
a = 10
b = 3

print("Qo'shish:", a + b)    # 13
print("Ayirish:", a - b)     # 7
print("Ko'paytirish:", a * b) # 30
print("Bo'lish:", a / b)     # 3.33...
print("Qoldiq:", a % b)      # 1

# Amaliy misol
narx = 50000
soni = 5
jami = narx * soni
print("Jami summa:", jami)   # 250000

💡 // — butun son bo'lish (10 // 3 = 3), ** — darajaga ko'tarish (2 ** 3 = 8)
